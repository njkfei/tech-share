# web(php)storm 常用快捷键

## 注释/解注

单行注释 `Ctrl + /`

多行注释 `Ctrl + Shift + /`

## 转到定义

光标指到方法上，按住 `Ctrl` + 鼠标单击，或者按 `Ctrl + B`

## 回到（向前、向后）原光标处

向后：`Ctrl + Alt + ←`

向前：`Ctrl + Alt + →`

## 搜索

全局搜索：`Ctrl + Shift + F`

当前搜索：`Ctrl + F`

## 替换

全局替换：`Ctrl + Shift + R`

当前替换：`Ctrl + R`

## 重构 

`Shift + F6`

## 查看某个方法被哪些地方调用
 
`Alt + F7`

## 打开一个已知文件
 
`Ctrl + Shift + N`

## 多行文本前后处理

按住 `Alt`
